螺旋千斤顶设计

螺旋千斤顶主要零件：螺杆、螺母、托杯、手柄、底座。
设计的 原始数据 ：最大起重量F=55KN、最大升起高度H=220mm。

Part1.螺杆的设计与计算

1.螺纹的牙型

选用矩形螺纹，采用内径对中，配合选H8/h8，在计算强度时不考虑螺纹的径向间隙。

2.螺杆的材料

选用45钢

3.螺杆的直径

螺杆工作时，同时受压力和转矩的作用，因此它的计算可近似按紧螺栓连接的计算公式求出螺纹内径，即 $$d_1 \geq \sqrt{{4 \times 1.3F} \over {\pi[\sigma]}}$$ 其中： $[\sigma] = {\sigma_S \over S}$
查阅资料: $[\sigma_S] = 353MPa$
取安全系数 $S = 3$
于是： $[\sigma] = {\sigma_S \over S} = $ $118MPa$
螺纹内径满足: $d_1 \geq \sqrt{{4 \times 1.3F} \over {\pi[\sigma]}} = $ $27.78mm$
螺距： $P = {d_1 \over 4} = $ $6.94mm$
参考矩形螺纹标准 ，取 $P = 6mm$
进而确定螺纹的工作高度： $h = {P \over 2} = $ $3mm$
螺纹大径： $d = d_1 + P = $ $33.78mm$
圆整后取： $d = 34mm$
因此， $d_1 = d - P =$ $28mm$

4.自锁验算

当量摩擦角 $\rho^\prime = \arctan{\mu \over \cos{\alpha \over 2}}$
由于螺杆和螺母的材料分别取45钢和锡青铜，因此取 $\mu = 0.10 (0.08 \sim 0.10)$
又因为矩形螺纹牙顶角 $\alpha = 0^\circ$
所以， $\rho^\prime = $ $5.711^\circ$
螺纹中径： $d_2 = {{d_1 + d} \over 2 } = $ $31mm$
螺纹中径升角： $\psi = \arctan {np \over {\pi d_2}} = $ $3.525^\circ$
满足 $\psi \leq \rho^\prime - 1^\circ$
因此，满足自锁条件，自锁可靠。

5.螺杆结构

螺杆上端用于支撑托杯和在其中插装手柄，因此需加大直径，其结构见图1。 手柄孔径$d_k$根据手柄直径$d_p$决定， $d_k \approx d_p + 0.5$（后面给出）。
为了便于切制螺纹，螺纹上端切有退刀槽，退刀槽的直径比$d_1$小$0.5mm$， 宽度为$1.5P=1.5 \times 6=9mm$。
所以退刀槽尺寸为： $9 \times \phi 27.5$
螺杆下端切有倒角，以便于螺杆旋入螺母，倒角尺寸为$C2$。为了防止工作时螺杆从螺母中脱出，在螺杆下端安置了钢制挡圈，挡圈用螺钉固定在螺杆端部。
挡圈直径为: $d_0 = d + 5 =$ 39mm
挡圈厚度$(6 \sim 12mm)$ 取8mm
螺钉直径为 $0.25d = $ 8.5mm
选取$GB/T68-2000 M10\times45$开槽沉头螺钉。

6.强度校核

螺杆工作时承受轴向压力$F$和转矩$T$的联合作用，根据第四强度理论求出其危险截面的当量应力,令其满足 $$\sigma_v = \sqrt{\sigma^2 + 3\tau^2} = \sqrt{({4F \over \pi d_1^3})^2 + 3({T \over 0.2d_1^3 })^2} \leq [\sigma]$$
其中，$T = F \cdot {d_2 \over 2} \tan({\psi + \rho^\prime}) = $ $113.42(N \cdot m)$
所以，$\sigma_v = \sqrt{({4F \over \pi d_1^3})^2 + 3({T \over 0.2d_1^3 })^2} = $ $99.90(MPa) \leq [\sigma] = 118MPa$
因此，强度符合强度要求。

Part2.螺母的设计与计算

1.材料

选用锡青铜材质，牌号：ZCuSn10Pb1

2.螺纹圈数

计算公式： $$Z \geq {F \over \pi d_2 h [p]}$$
其中螺纹中径$d_2=29mm$，螺纹牙高度$h=3mm$，螺纹牙表面许用压强$[p]$取$25MPa(18 \sim 25MPa)$ 。代入上式，得 $$Z \geq {F \over \pi d_2 h [p]} = $$ $7.52$
因此，取$Z = $ $8$
考虑到退刀槽的影响，实际螺纹圈数:$Z^\prime = Z + 1.5 = $ $9.5$
螺母高:$H^\prime = Z^\prime P = $ $57mm$

3.螺纹其它尺寸(见图2)

$a = {H^\prime \over 3} = $ $19mm$
$D \leq 1.6d = $ $ 1.6 \times 34 = 54.4mm$ 取 $D = 55mm$
$D_1 \leq 1.3D = $ $ 1.3 \times 55 = 71.5mm$ 取 $D_1 = 72mm$

4.螺母与底座的配合

螺母压入底座上的孔内，圆柱面配合采用$H8/n7$。 螺母下端和底座孔上端均加工出倒角，倒角尺寸为$C2$。 同时安置紧定螺钉防止螺母转动，紧定螺钉选用$GB/T65$的开槽圆头螺钉。

Part3.螺杆稳定性计算

螺杆稳定性条件为:$$S_{sc} = {F_cr \over F}$$ 式中，$S_s$为螺杆稳定性计算安全因数;
$S_s$为螺杆稳定性安全因数$(S_s = 2.5 \sim 4)$；
$F_{cr}$为螺杆的临界载荷。
计算螺杆的长细比$\lambda$:$$\lambda = {\mu \cdot l \over i}$$ 其中，长度系数：$\mu = 2$; 螺杆最大受压长度：$$l = H + {H^\prime \over 2} + 1.5d = $$ $299.5mm$;
螺杆危险截面惯性半径：$i = {d_1 \over 4 }= $ $7$
所以，$\lambda = $ $85.87$
对于$\lambda < 85$淬火钢 $F_{cr} = {480 \over {1+0.0002\gamma^2}} \cdot {\pi d_1^2 \over 4} = $ $119.93(KN)$
进而可得$S_{sc} = {F_{cr} \over F} = $ $2.65$
由于螺杆稳定性安全因数$S_s = 2.5 \sim 4$
所以螺杆稳定性合格。

Part4.托杯的设计与计算

1.材料

选用Q235碳钢，模锻制造。

2.杯托结构（见图3）

取壁厚$\delta = 10mm$
托杯高$（1.5 \sim 1.8d）$,
所以杯托高 $= 1.6d = $ $51.2mm$ 取 $52mm$
直径$D_2 = 0.6d = $ $20.4mm$ 取 $D_2 = 21mm$
$D_3 = 2.4d = $ $81.6mm$ 取 $D_3 = 82 mm$
为了使其与重物接触良好，防止重物滑动，在托杯上表面制出切口和沟纹。 为了防止托杯从螺杆端部脱落，在螺杆上端应装置挡板，挡板尺寸$\phi 30 \times \phi 6 \times 3$， 螺钉选用$GB/T5783-2000M5 \times 25$开槽圆角头螺钉。 直径$D_2$的配合为$H11/a11$。
当螺杆转动时，托杯和重物都不转动。 因此，在起重时，托杯底部与螺杆的接触面间有相对滑动。 为了避免过快磨损，一方面需要润滑，另一方面要限制接触面间的压力，取其许用压力为 $$[p] = 20MPa(15 \sim 25MPa)$$ 由此可确定托杯直径$D_4 \leq \sqrt{{4F \over (\pi [P] )}+D_2^2} = $ $62.79mm$ 取 $D_4 = 63mm$

Part5.手柄的设计与计算

1.材料

$Q235$碳钢

2.手柄长度$L_p$

扳动手柄的转矩$KL_p$必须克服螺纹副的螺纹力矩$M_1$和螺杆与托杯间的摩擦力矩$M_2$，即 $$KL_p \geq M_1 + M_2$$ 则$L_p \geq {{M_1 + M_2} \over K}$
式中，$K$为施加于手柄上的臂力$(150 \sim 250 N)$，为满足大多数人的使用， 设计中取$K = 200N$ 。
$$M_1 \approx F \cdot \tan({\psi + \rho^\prime}) \cdot {d_2 \over 2} = $$ $138.62(N \cdot m)$ $$M_2 = 0.25(D_2 + D_4) \mu F= $$ $69.3(N \cdot m)$
式中，$\mu = 0.06(0.05 \sim 0.08)$。
所以，手柄计算长度满足$L_p \geq {{M_1 + M_2} \over K} = $ $1039.6mm$
考虑到螺杆头部尺寸及手握长度的影响，手柄实际长度为 $L_w = Lp + 0.5D_4 + 100 = $ $1171.1mm$
$L_w$不应超过千斤顶的最小高度，所以将手柄实际长度改为$352mm$，另外加套筒，套筒长$800mm$。

3.手柄直径

计算公式： $$d_p \geq \sqrt[3]{KL_p \over (0.1[\sigma]_b)}$$ 式中，$[\sigma]_b$为材料的许用弯曲应力，对于$Q235$的手柄， $$[\sigma]_b = 120MPa$$ 所以,$d_p \geq \sqrt[3]{KL_p \over (0.1[\sigma]_b)} = $ $25.88mm$ 取 $d_p = 25.88mm$
则螺杆孔径$d_k = d_p +0.5 =$ $26.38mm$

4.手柄结构

手柄插入螺杆上端的孔中，为防止手柄从孔中滑出，在手柄两端应加上挡环，并用螺钉固定，其结构见图4。
固定螺钉选用$GB/T68-2000 M8 \times 20$开槽沉头螺钉。

Part6.底座的设计与计算

1.材料

$HT100$铸铁。

2.尺寸设计

见图5，取壁厚$\delta = 10mm$，锥度按照$1:5$设计。
各部分尺寸为：
$H_1 = H + 20 = $ $240mm$
$D_5 = D + 6 = $ $61mm$
$D_6 = D_5 + {H_1 \over 5} = $ $109mm$
$D_7 = \sqrt{{4F \over \pi [\sigma]_p} + D_6^2} = $ $199.73mm$
式中，$[\sigma ]_p$为底座下枕垫物的许用挤压应力。以下枕垫物为木材考虑，$[\sigma ]_p = 2.5MPa$。
因为$D_8 \geq D_1$
所以，取$D_8 = 72mm$

Part7.千斤顶的效率

计算公式： $$\eta = {FP \over {2\pi (T_1 + T_2)}}$$ 一般$\eta < 0.3$
本次设计中$\eta = 0.253 < 0.3$
因此，设计效率符合要求。
